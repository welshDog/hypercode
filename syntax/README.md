# HyperCode Syntax Definitions

This directory contains the formal grammar and syntax definitions for HyperCode.

## Files
- `HyperCode.g4`: ANTLR4 grammar file (primary source of truth)
- `syntax_guide.md`: Human-readable syntax guide
- `visual_patterns.md`: Documentation of visual/spatial syntax elements

## Note
The parser implementation is currently located in `../hypercode/parser`.
