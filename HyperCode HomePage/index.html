See It in Action

Here's how you can create a quantum Bell pair in HyperCode:


@quantum_function: bell_pair() -> Bits
    @circuit: c
        @qubits: q[2]
        @register: r[2]

        # Put the first qubit in a superposition
        @hadamard: q[0]

        # Entangle the qubits
        @cnot: q[0], q[1]

        # Measure the results
        @measure: q[0] -> r[0]
        @measure: q[1] -> r[1]

    @return: r

Get Started with HyperCode

The HyperCode interpreter is a command-line tool. You can run your first program in a few steps.


# 1. Clone the repository
git clone https://github.com/welshDog/THE-HYPERCODE.git

# 2. Navigate to the project directory
cd THE-HYPERCODE/THE-HYPERCODE

# 3. Run the Bell Pair example
python -m hypercode run examples/bell_pair.hc
